# Junubin Sanitation Services

A pure static website for Junubin Sanitation Services.

## Technical Details
- Pure HTML, CSS, and JavaScript
- No build step required
- Uses Tailwind CSS via CDN for styling
- Fully responsive design
- Ready to be hosted on GitHub Pages

## Project Structure

```
├── js/

├── assets/
│   ├── images/
│   └── icons/
└── README.md
```

## Deployment
This website can be deployed directly to GitHub Pages:
1. Upload all files to a GitHub repository
2. Go to Settings > Pages
3. Select "Deploy from a branch"
4. Choose the `main` branch and `/ (root)` folder
5. Save and wait for the deployment to complete
